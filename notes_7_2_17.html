Intro to Databases

  • Relational
    • Information stored in tables
    • Column corresponds to a field
    • Row corresponds to a single record
    • Types : mySQL, PosGRES, Oracle
  • Non-relational
    • NoSQL
    • Redis
    • Mongo
    • Couch
    • Neo4j
  • Schema : tells Database when to yell at you
  • primary key : unique to each record
  • foreign key : primary key from different table

    • What is SQL?

    • DDL : Describing Data
    • DML : CRUD
      • C: Create/ Insert
      • R: Select
      • U: Update
      • D: Delete, Drop, Truncate
    • ALTER : command to modify existing objects
    • ADD : command to add objects
